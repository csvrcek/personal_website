Magic 8



Description

Magic 8 is an iOS application inspired from the classic Magic 8-Ball toy. This app gives the user an easy-to-use interface that provides a button and a text view on the main screen. When the button is pressed, a fortune is randomly generated from an array of fortunes. That fortune is then displayed on the text view for the user to see. During this project, I learned about Swift code, the syntax of the language, how to implement it for the logic of an app, and how to work with Xcode.
