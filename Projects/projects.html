School Projects


Braking Coefficient

Braking Coeffient was one of the C++ projects I made during my Engineering 101 class at Michigan. It is a program that can be used to calculate the proper braking coefficient a pilot must apply when landing an airplane, given a certain initial velocity, drag coefficient, and runway length.

Runway Material

This program was another C++ project developed during Engineering 101. The purpose of this program was to take in some information about some materials, like Concrete and Steel for example, and output a report that includes information about whether or not the material will be useful for certain situations.

gUber

gUber, the final C++ project in Engineering 101, encorporated everything about C++ we had learned throughout the second half of the semester. The program dealt with the Travelling Salesman Problem. The idea behind this project was that there is an intergalactic delivery service, and we needed to deliver to every planet in a given grid. No matter the grid of planets given, gUber had to produce step-by-step directions for each planet, including a beginning planet and an ending planet.
